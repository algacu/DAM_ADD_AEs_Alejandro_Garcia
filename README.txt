# Actividades evaluables de Acceso a Datos.

Segundo año CFGS Desarrollo de Aplicaciones Multiplataforma.
Lenguaje de programación: JAVA (IDE Eclipse - Oracle).

AE1 - Introducción a la programación de apps en Java para el acceso a ficheros y bases de datos relacionales (MySQL).
AE2 - Programación de aplicaciones con el uso de Hibernate y mapeo objeto-relacional en apps de Java orientadas a objetos.
AE3 - Programación de apps para el acceso a datos a través de objetos XML y JSON en bases de datos no relacionales (MongoDB, Firebase).
AE4 - Programación de aplicaciones para el acceso a fuentes de datos remotas, con el uso de arquitectura API-REST.
